# أداة التحسين الجنائي للصور
## Forensic Image Enhancement Tool v2.0 — GitHub Actions Edition

---

## طريقة الاستخدام اليومية

### الخطوة 1: ارفع الصورة
ضع صورتك في مجلد `input/` في الـ repository

### الخطوة 2: شغّل الأداة
اذهب إلى:
```
Actions → Forensic Image Enhancement → Run workflow
```
أدخل رقم القضية واضغط **Run workflow**

### الخطوة 3: حمّل النتائج
بعد 5-10 دقائق اضغط على **forensic-results** لتحميل:
- الصورة المحسنة بـ Real-ESRGAN
- الوجوه المحسنة بـ GFPGAN
- صورة المقارنة قبل/بعد
- التقرير الجنائي HTML كامل
- بيانات JSON الخام

---

## هيكل الـ Repository

```
Osama/
├── input/          ← ضع صورك هنا
├── output/         ← النتائج (تلقائي)
├── forensic.py     ← الأداة الرئيسية
└── .github/
    └── workflows/
        └── enhance.yml  ← GitHub Actions
```

---

## التقرير الجنائي يحتوي:
- ✅ MD5 + SHA-256 للتحقق من سلامة الملف
- ✅ أبعاد الصورة قبل وبعد
- ✅ تحليل حدة الصورة (Laplacian Variance)
- ✅ مستوى الضوضاء والسطوع والتباين
- ✅ بيانات EXIF المضمنة
- ✅ صورة مقارنة قبل/بعد
- ✅ نسبة التحسين المئوية
